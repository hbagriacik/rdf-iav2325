Beispiele mit Attributselektoren

Eigentlich kann man fast alles mit Attribut-Selektoren angesprechen.

1 Links mit einem bestimmten Filetype

PDF-File PDF Word-File DOC Exel-File XLS

Die Formatierung der obenstehenden Links erfolgt über den Dateityp in der Linkadresse: also 'pdf', 'doc' bzw. 'xls'.

2 Attribute deren Wert mit einem bestimmten Wort enden

In diesem Beispiel wird das 'title' Attribut angesprochen, bei dessen Wert am Ende das Wort 'Nürnberg' steht.

Stadtportal Nürnberg Verkehrsstreit in Nürnberg eskaliert Nürnberg: 17-Jähriger zerstört Rettungswagen

Achtung: Gross- bzw. Kleinschreibung muss beachtet werden, ansonsten funktioniert es nicht.

3 Irgendwo im Attribut ein bestimmtes Wort

In diesem Beispiel wird das 'title' Attribut angesprochen, bei dem an beliebiger Stelle im Wert das Wort 'KTM' vorkommt.

Motorhall Zubehörpakete für Adventure Bikes von Yamaha und KTM Shop Motorcycle Parts

4 Attribute deren Wert mit einem bestimmten Wort beginnen

Hier wird das Attribut 'href' angesprochen. Beginnt die Adresse mit 'http://' wird ein Icon angezeigt. So können zum Beispiel alle externen Links mit Attribut-Selektoren angesprochen werden

http://irgendwohin.weg

5 Ansprechen eines speziell definierten Attributes

Selektoren auf das Attribut 'type' eines Input Elements ersetzen Klassen, die vorher gebraucht wurden um den verschiedenen Typen (text, radio, checkbox) gerecht zu werden.

Für Formulare eignen sich Attribut-Selektoren besonders. Man kann sie nach dem title Attribut selektieren, oder wie im folgenden Beispiel den Input-Type direkt ansprechen. Man kann diese Anwendungen auch problemlos kombinieren oder nur spezielle Inputs vom Typ text ansprechen, in denen ein bestimmtes Wort im title vorkommt. Die Definition kann man dann noch vom Kontext abhängig machen, um die Darstellung auf einen spezifischen Bereich einzuschränken.
